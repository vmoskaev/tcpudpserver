TCP и UDP сервер в одном консольном приложении C++.

Также в каталоге client реализован тестовый клиент C++ для проверки работы сервера.Запускается по 5 клиентов TCP и UDP, которые отправляют сообщения серверам. По окончании отправки сообщений и получения ответа от сервера клиенты завершаются. По окончанию работы последнего клиента TCP, UDP завершается соответствующий сервер по команде /shutdown.

Все реализовано на стандартной библиотеке STL, без использования сторонних библиотек.Для получения сообщений везде используется epoll. Обработка сообщений запускается в отдельном потоке через std::async.
